# InfoWorks WS Pro

Welcome to the InfoWorks WS Pro Scripts repository! This repository contains open source Ruby, SQL, VBScript, and other scripts for building and using [InfoWorks WS Pro](https://help.autodesk.com/view/IWWSPRO/2026/ENU/) models.

## Folder Structure

| Folder | Contents |
|--------|----------|
| [01 Ruby](./01%20Ruby/) | Ruby scripts (UI and Exchange) |
| [02 SQL](./02%20SQL/) | SQL queries for network selection and manipulation |
| [03 VBScript](./03%20VBScript/) | VBScript automation and integration tools |
| [04 End-to-End Workflows](./04%20End-to-End%20Workflows/) | Multi-step pipelines combining Ruby, Python, and batch automation |
