# Sample Database Ruby Scripts

- Auto Manhole Labels.rb
- Auto-profiling.rb
- Bulk delete of scenarios.rb
- Connect subcatchment to nearest node.rb
- Design flows from population.rb
- Earthwork estimator for pipes and storage.rb
- Find the longest upstream path of the selected node.rb
- Select bifurcation nodes.rb
